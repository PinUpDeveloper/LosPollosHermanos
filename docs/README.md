# Docs

Текстовая документация проекта хранится в этой папке.


- `architecture.md` — архитектура и взаимодействие компонентов
- `demo-flow.md` — сценарий демо для жюри
- `pitch-notes.md` — формулировки для питча и позиционирования
- `api.md` — ключевые backend endpoints и контракты
- `roadmap.md` — технический roadmap реализации
- `feature-roadmap.md` — продуктовый roadmap и demo-фичи
